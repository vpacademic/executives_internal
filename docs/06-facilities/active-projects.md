## Project Entry Standard

Each project entry should include:

- Project name and responsible role.
- Approved folder link.
- Current phase or next decision point.
- Key agreement, budget, or Council/HFC dependency if applicable.
- Last confirmed update date.

Do not use this page for draft negotiations, private staff-only material, or
documents that include sensitive operational details.
